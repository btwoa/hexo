在模拟连接服务器的过程中，为了制造紧张的氛围感，当ip被追踪到会给予警告，第三次警告后将强制停止游戏。

攻略

m1

要求

  • 获取并安装 PortScannerW32.exe
  • 删除日志 192.168.010.001

步骤

  • ping ip地址
  • telnet[Telnet] 连接

  • 用户名密码登录

  • cd hackingtoolz
  • get file[FileLeecher] PortScannerW32.exe

  • cd telnetlogs
  • del 192.168.010.001

  • 断开连接

  • cd downloads
  • PortScannerW32.exe 安装
