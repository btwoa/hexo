## 查看执行策略

 获取当前的执行策略

```
Get-ExecutionPolicy
```

![](https://ovo.btwoa.com/img/webp/202202021428106.webp)

获取所有执行策略和有效执行策略

```
Get-ExecutionPolicy -List
```

![](https://ovo.btwoa.com/img/webp/202202021433054.webp)

## 设置执行策略

设置执行策略为要求远程签名，范围为当前用户

```
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```
